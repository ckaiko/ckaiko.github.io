World Explorer

A 2D procedurally-generated exploration game built in Java

Java Game Dev Procedural Generation 2D Graphics

Overview

World Explorer is a 2D tile-based exploration game written entirely in Java. The game procedurally generates unique worlds each playthrough using randomized algorithms, creating explorable environments with rooms, hallways, and interactive elements.

Built as part of the BYOW (Build Your Own World) project, it demonstrates core concepts in procedural generation, data structures, and 2D rendering.

Key Features

  • Procedurally generated 2D worlds with unique layouts on each run
  • Tile-based rendering engine for rooms, corridors, and obstacles
  • Player movement and world interaction mechanics
  • Seed-based generation for reproducible world states
  • Save and load game state functionality

Challenges & What I Learned

This project was a deep dive into procedural content generation, graph algorithms for room connectivity, and managing game state in Java. It also strengthened my understanding of object-oriented design patterns and tile-based rendering.

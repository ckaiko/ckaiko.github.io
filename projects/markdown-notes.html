Markdown Notes

A lightweight, local-first note-taking app

Rust Tauri Markdown

Overview

Markdown Notes is a desktop app that stores notes as plain .md files on your filesystem. It renders Markdown in real-time, supports full-text search across all notes, and has a tagging system for organization.

I built it because I wanted something faster and simpler than Notion that didn't require an internet connection.

Key Features

  • Live Markdown preview with syntax highlighting
  • Full-text search powered by tantivy (Rust search library)
  • Tag-based organization with sidebar filtering
  • All data stored as plain .md files — no vendor lock-in
  • Cross-platform (macOS, Windows, Linux) via Tauri

What I Learned

This was my first Rust project and my introduction to Tauri as a lightweight alternative to Electron. I learned about Rust's ownership model, how to bridge Rust and JavaScript, and how to build a search index from scratch.

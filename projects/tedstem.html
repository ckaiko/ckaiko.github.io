TedStem

AI Hacks 2024 hackathon project

TypeScript Python AI / ML Hackathon

Overview

TedStem is a project built for AI Hacks 2024, combining TypeScript and Python to create an AI-powered application. The dual-language architecture leverages TypeScript for the application layer and Python for the AI/ML backend processing.

Key Features

  • Built during AI Hacks 2024 hackathon
  • Dual-language architecture: TypeScript (48.9%) and Python (46.7%)
  • AI/ML pipeline integrated into the application workflow
  • Rapid prototyping under hackathon constraints

What I Learned

This hackathon pushed me to rapidly prototype an AI-powered solution under tight time constraints. Working across TypeScript and Python in a single project taught me about managing polyglot codebases and integrating ML models into application pipelines.

BeMeal

A BeReal-inspired food diary app for capturing daily meals with photos

Swift SwiftUI UIKit iOS MVVM FileManager In Progress

Overview

BeMeal is an iOS food diary app inspired by BeReal that lets users visually document their daily meals, snacks, and drinks through photos. Each entry is captured with a timestamp, an optional caption, and a color-coded category, making it easy to look back on eating habits over time.

The app features a reactive SwiftUI interface built around a single-store architecture, with a Today feed for real-time entries, a History view for browsing past days, and BeReal-style photo cards with gradient overlays for a polished, social-app feel.

Key Features

  • Snap photos via camera or pick from the photo library using a bridged UIKit image picker
  • Categorize entries as Meal, Snack, or Drink with color-coded pill buttons and custom icons
  • Today feed displaying a scrollable timeline of the day’s entries with a daily summary bar
  • History view with calendar-style browsing, day summaries, and drill-down into past entries
  • Persistent local storage using JSON serialization and JPEG images saved to the Documents directory
  • Long-press context menus for entry deletion with automatic image file cleanup

Challenges & What I Learned

Building BeMeal deepened my understanding of modern SwiftUI patterns including the @Observable macro, reactive state management with a single source of truth, and bridging UIKit components into SwiftUI via UIViewControllerRepresentable. I also gained hands-on experience with file-based persistence using FileManager, image compression and lazy loading, and designing reusable view components for a clean, maintainable codebase.

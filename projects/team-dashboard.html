Team Dashboard

Internal project tracker for sprint planning and team capacity

Vue D3.js PM Tools REST API

Overview

Team Dashboard is an internal tool I designed and built for our engineering team. It pulls data from Jira and GitHub to give PMs and leads a single view of sprint progress, velocity trends, and individual capacity.

The goal was to replace the "How are we tracking?" Slack questions with a living dashboard that updates itself.

Key Features

  • Velocity charts with rolling 6-sprint averages (D3.js)
  • Sprint board with drag-and-drop story reordering
  • Team capacity planner with PTO and focus-time blocking
  • GitHub PR status integration (open, reviewed, merged)
  • Jira sync every 5 minutes via background workers

What I Learned

This was my first real Vue project and my introduction to D3.js for data visualization. I also learned a ton about OAuth flows for third-party API integrations and the challenges of keeping cached data fresh.

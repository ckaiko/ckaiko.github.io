MediaPlaylistCreator

Movie & Song Playlist Recommender powered by AI embeddings

Python Streamlit PostgreSQL OpenAI Spotify API Docker In Progress

Overview

MediaPlaylistCreator is a full-stack Python application that generates personalized movie and song playlist recommendations. It pulls viewing history from Letterboxd via RSS, matches musical moods using the Spotify API, and leverages OpenAI embeddings for semantic similarity matching.

The project features a Streamlit dashboard for exploring recommendations, a PostgreSQL backend for storing user data and embeddings, and Docker for reproducible deployments.

Key Features

  • Letterboxd RSS integration to import movie watch history automatically
  • Spotify API integration for song metadata, audio features, and playlist generation
  • OpenAI embeddings for semantic similarity—matching movie moods to songs
  • PostgreSQL database for persistent storage of user profiles and recommendation history
  • Interactive Streamlit dashboard for browsing and refining recommendations
  • Fully containerized with Docker for easy setup and deployment

Challenges & What I Learned

This project deepened my understanding of embedding-based recommendation systems, API rate limiting strategies, and building interactive data apps with Streamlit. It also gave me hands-on experience with vector similarity search and PostgreSQL performance tuning.

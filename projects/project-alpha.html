Project Alpha

A full-stack web app with real-time data syncing

React Node.js WebSockets JWT PostgreSQL

Overview

Project Alpha is a full-stack web application built with Node.js and React. It features real-time data syncing via WebSockets, user authentication with JWT tokens, and a responsive dashboard for monitoring live metrics.

The project started as a weekend hack and grew into a production tool used by a small team of 12. It handles ~500 concurrent WebSocket connections with sub-100ms latency.

Key Features

  • Real-time dashboard with live-updating charts (D3.js + WebSockets)
  • JWT-based authentication with refresh token rotation
  • PostgreSQL database with Prisma ORM
  • Responsive design that works on mobile and desktop
  • Automated CI/CD pipeline with GitHub Actions

What I Learned

This project taught me a lot about managing WebSocket connections at scale, structuring a monorepo for frontend and backend code, and writing meaningful integration tests. It was also my first time deploying to AWS with Terraform.

CSM 61A Worksheets

LaTeX worksheets for UC Berkeley's CS Mentors program

LaTeX Python Teaching CS Education

Overview

A collection of teaching worksheets created for Computer Science Mentors (CSM) at UC Berkeley, supporting CS 61A — the introductory computer science course. The worksheets are written in LaTeX and cover core topics in Python programming, recursion, higher-order functions, and data abstraction.

Forked from the CSM Berkeley repository, this project includes my contributions and customizations to the worksheet content used in weekly small-group mentoring sessions.

Key Features

  • Comprehensive LaTeX worksheets covering CS 61A curriculum topics
  • Problems and solutions for Python fundamentals, recursion, trees, and linked lists
  • Designed for small-group mentoring sessions (4–6 students per section)
  • Collaborative contributions to the CSM Berkeley open-source repository

Challenges & What I Learned

Contributing to these worksheets sharpened both my LaTeX skills and my ability to explain complex CS concepts clearly. Mentoring students through these materials reinforced my own understanding while developing my teaching and communication abilities.

Freelance Web Developer

2019 – 2020

Overview

Before starting university, I freelanced building websites and landing pages for small businesses. This is where I learned the fundamentals of web development and, more importantly, how to work with real clients.

What I Did

  • Designed and built 8+ websites for local businesses (restaurants, salons, photographers)
  • Created responsive landing pages for marketing campaigns
  • Set up hosting, domains, and basic analytics for clients
  • Managed client expectations, timelines, and revision rounds

What I Learned

Freelancing taught me that building software is only half the job — the other half is communication. I learned how to scope work, set realistic deadlines, handle feedback gracefully, and ship on time when the stakes are real.
